# Frontend UI Quick Start

Environment
- Copy .env.example to .env and set:
  - REACT_APP_API_BASE_URL=http://localhost:3001

Run
- npm install
- npm start
- Open http://localhost:3000

Notes
- Ensure the backend API is running on the port specified in REACT_APP_API_BASE_URL.
- Update CORS in the backend via FRONTEND_ORIGIN if you change frontend URL/port.
